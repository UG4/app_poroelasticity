# Poroelasticity 

## Governing equations
The governing equations read:
```math 
\partial_t (\Phi p + \alpha \nabla \cdot  u) + \nabla \cdot [-\kappa \nabla p] = f
```

```math
\nabla \cdot [\sigma - \alpha p \mathcal I] = f
```
